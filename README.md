# TanjiroPassCheck

Password strength checker with RockYou detection.

Author: tanjironezuko1900

## Features

- Password strength rating
- RockYou password detection
- Hidden password input
- Supports Ubuntu and Debian

## Installation

git clone https://github.com/tanjionezuko1900/TanjiroPassCheck.git

cd TanjiroPassCheck

sudo chmod +x install.sh

sudo ./install.sh

## Usage

chmod +x tanjiropasscheck.sh

./tanjiropasscheck.sh
